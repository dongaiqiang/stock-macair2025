## 📖 详细功能说明

### 功能 1：查看股票 K 线图

**操作步骤：**

1. 在首页顶部的「股票代码」输入框中输入股票代码（如：`600519`）
2. 选择开始日期和结束日期（如：`2024-01-01` 至 `2024-12-31`）
3. 点击「获取数据」按钮
4. 等待数据加载完成后，即可看到 K 线图

**支持的技术指标：**
- MA（移动平均线）：MA5、MA10、MA20、MA60
- EMA（指数平均线）：EMA12、EMA26
- MACD（平滑异同移动平均线）
- RSI（相对强弱指标）
- KDJ（随机指标）
- BOLL（布林带）
- OBV（能量潮）
- ATR（平均真实波幅）
- ADX（平均趋向指数）

**热门股票快捷选择：**
- `600519` 贵州茅台
- `000858` 五粮液
- `601318` 中国平安

---

### 功能 2：全部策略回测

**操作步骤：**

1. 输入股票代码和日期范围
2. 点击「全部回测」按钮
3. 等待回测完成（约 10-30 秒）
4. 切换到「策略回测结果」标签页查看结果

**回测结果展示：**
- 总收益率（%）
- 最大回撤（%）
- 总交易次数
- 胜率（%）
- 最终资金

**支持的 7 种策略：**

| 策略 ID | 策略名称 | 说明 |
|--------|---------|------|
| ma | 双均线策略 | MA5/MA20 金叉买入，死叉卖出 |
| rsi | RSI 超买超卖 | RSI<30 买入，RSI>70 卖出 |
| boll | 布林带突破 | 突破下轨买入，突破上轨卖出 |
| macd | MACD 金叉死叉 | MACD 金叉买入，死叉卖出 |
| turtle | 海龟交易法 | 突破 N 日高点买入 |
| mean_reversion | 均值回归 | 价格低于布林带下轨买入 |
| tail_breakout | 尾盘突破 | 尾盘突破买入，次日卖出 |

### 功能 3：单策略回测（自定义参数）

**操作步骤：**

1. 在「单策略回测」区域，从下拉菜单选择策略
2. 配置策略参数（展开显示参数配置表单）
3. 点击「运行单策略回测」按钮
4. 查看回测结果

**各策略可配置参数：**

#### 双均线策略 (ma)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| fast_period | 快线周期 | 5 | 1-60 |
| slow_period | 慢线周期 | 20 | 1-120 |
| stop_loss | 止损比例 (%) | 0 | 0-100 |
| take_profit | 止盈比例 (%) | 0 | 0-200 |

#### RSI 策略 (rsi)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| rsi_period | RSI 周期 | 14 | 2-50 |
| rsi_upper | 超买线 | 70 | 50-95 |
| rsi_lower | 超卖线 | 30 | 5-50 |

#### 布林带策略 (boll)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| period | 周期 | 20 | 5-60 |
| devfactor | 标准差倍数 | 2.0 | 0.5-4 |

#### MACD 策略 (macd)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| fast | 快线周期 | 12 | 2-50 |
| slow | 慢线周期 | 26 | 10-100 |
| signal | 信号线周期 | 9 | 2-30 |

#### 海龟交易策略 (turtle)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| entry_period | 入场周期 | 20 | 5-60 |
| exit_period | 出场周期 | 10 | 3-30 |

#### 均值回归策略 (mean_reversion)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| period | 周期 | 20 | 5-60 |
| std_dev | 标准差倍数 | 2.0 | 0.5-4 |

#### 尾盘突破策略 (tail_breakout)
| 参数名 | 说明 | 默认值 | 范围 |
|-------|------|-------|------|
| lookback | 回顾周期 | 20 | 5-60 |
| breakout_rate | 突破阈值 (%) | 98 | 90-100 |

### 功能 4：API 接口测试

后端提供完整的 RESTful API，可用于集成或其他用途。

#### API 端点列表

**1. 获取策略列表**
```bash
curl http://localhost:8000/api/strategies
```

**2. 获取股票数据**
```bash
curl -X POST http://localhost:8000/api/stock/data \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"600519","start_date":"2024-01-01","end_date":"2024-12-31"}'
```

**3. 获取 K 线图数据（含技术指标）**
```bash
curl -X POST http://localhost:8000/api/stock/chart-data \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"600519","start_date":"2024-01-01","end_date":"2024-12-31"}'
```

**4. 单策略回测**
```bash
curl -X POST http://localhost:8000/api/backtest \
  -H "Content-Type: application/json" \
  -d '{
    "stock_code":"600519",
    "start_date":"2024-01-01",
    "end_date":"2024-12-31",
    "strategy":"ma",
    "params":{"fast_period":5,"slow_period":20}
  }'
```

**5. 全部策略回测**
```bash
curl -X POST http://localhost:8000/api/backtest/all \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"600519","start_date":"2024-01-01","end_date":"2024-12-31"}'
```

**6. 多股票回测**
```bash
curl -X POST http://localhost:8000/api/backtest/multi \
  -H "Content-Type: application/json" \
  -d '{
    "stock_codes":["600519","000858"],
    "start_date":"2024-01-01",
    "end_date":"2024-12-31",
    "strategy":"ma"
  }'
```

**7. 机器学习预测**
```bash
curl -X POST http://localhost:8000/api/ml/predict \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"600519","start_date":"2024-01-01","end_date":"2024-12-31"}'
```

## ❓ 常见问题

### Q1: 获取数据失败怎么办？
A: 检查网络连接，确保 baostock 数据源可访问。可尝试更换股票代码或日期范围。

### Q2: 回测结果显示为空？
A: 回测需要时间处理，请等待进度条完成。如果数据量过大（>1 年），可能需要 30-60 秒。

### Q3: 如何修改策略参数？
A: 在「单策略回测」区域选择策略后，下方会自动展开参数配置表单。

### Q4: 移动端如何使用？
A: 系统已适配移动端，在手机浏览器访问同一地址即可。

### Q5: 如何查看详细的 API 文档？
A: 访问 `http://localhost:8000/docs` 可查看 Swagger/OpenAPI 文档。
