## 遇到的问题及解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| NaN 导致 JSON 序列化失败 | pandas 中技术指标计算产生 NaN | 使用 `.where(df.notna(), None)` 替换 |
| 后端导入模块失败 | Python 路径配置问题 | 使用绝对路径 `ROOT_DIR` |
| toFixed 报错 | undefined 值未处理 | 添加 `?? 0` 空值保护 |
| API 监听测试失败 | 响应时机问题 | 非关键，可优化测试逻辑 |
| 回测收益率显示 0% | 小数转百分比未乘 100，精度不足 | 添加 `* 100` 转换，小数值用 4 位小数显示 |

## Bug 修复详情

### 问题：回测收益率显示为 0%

**原因**：
1. 后端返回的是小数格式（如 `0.001` 表示 `0.1%`）
2. 前端未乘以 100 直接显示
3. 小幅收益率（如 `0.00079`）用 2 位小数显示变成了 `0.00%`

**修复方案**：
```typescript
// 后端返回的小数转百分比
const safeReturn = (result.total_return ?? 0) * 100;
const safeDrawdown = (result.max_drawdown ?? 0) * 100;

// 根据数值大小动态调整精度
const returnPrecision = Math.abs(safeReturn) < 0.1 ? 4 : 2;
```

**修复效果**：
```
修复前：+0.00%
修复后：+0.0792%  (4 位小数)
```
