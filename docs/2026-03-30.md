### 1. 修复技术指标除零 Bug

**文件**：`indicators/indicators.py`

**问题**：
- RSI 计算时 `rs = avg_gain / avg_loss` 出现除零错误
- KDJ 计算时 `(high_n - low_n)` 为 0 时除零
- ADX 计算时 `(plus_di + minus_di)` 为 0 时除零

**修复方案**：
```python
# RSI
rs = avg_gain / avg_loss.replace(0, np.nan)

# KDJ
denominator = (high_n - low_n).replace(0, np.nan)
raw_k = 100 * (close - low_n) / denominator

# ADX
dx = 100 * np.abs(plus_di - minus_di) / (plus_di + minus_di).replace(0, np.nan)
```

### 2. 请求缓存实现

```python
_stock_data_cache = {}

async def get_from_cache(key: str, ttl: int = 300) -> Optional[dict]:
    if key in _stock_data_cache:
        cache_entry = _stock_data_cache[key]
        if time.time() - cache_entry["timestamp"] < ttl:
            logger.info(f"缓存命中：{key}")
            return cache_entry["data"]
        else:
            del _stock_data_cache[key]
    return None
```

### 3. 策略参数百分比转换

```typescript
const params: Record<string, any> = {};
switch (strategy) {
  case "turtle":
    params.entry_period = formData.entry_period;

    break;
  case "mean_reversion":
```
