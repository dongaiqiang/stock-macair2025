## 系统概述

本系统是一个量化股票交易回测平台，提供以下功能：

- 📈 **K 线图查看** - 支持 A 股股票历史数据可视化
- 📊 **策略回测** - 7 种经典交易策略一键回测
- 📱 **响应式设计** - 支持电脑、手机、平板访问
- 🤖 **机器学习** - 基于 LightGBM 的股价预测

## 功能操作指南

### 1. 查看 K 线图

**操作步骤：**

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 输入股票代码 | 如：`600519`（贵州茅台） |
| 2 | 选择开始日期 | 如：`2024-01-01` |
| 3 | 选择结束日期 | 如：`2024-12-31` |
| 4 | 点击「获取数据」按钮 | 蓝色按钮 |
| 5 | 查看 K 线图 | 在页面下方显示 |

**预期结果：**
- 显示蜡烛图（红色=上涨，绿色=下跌）
- 可拖动缩放、左右滑动查看历史

---

### 2. 运行策略回测

**操作步骤：**

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 输入股票代码和日期 | 同上 |
| 2 | 点击「全部回测」按钮 | 绿色按钮 |
| 3 | 等待回测完成 | 约 10-30 秒 |
| 4 | 切换到「策略回测结果」标签 | 查看对比 |

**7 种策略说明：**

| 策略 | 原理 | 适用场景 |
|------|------|----------|
| 双均线 | MA5/MA20 金叉买入，死叉卖出 | 趋势行情 |
| RSI 超买超卖 | RSI<30 买入，RSI>70 卖出 | 震荡行情 |
| 布林带突破 | 突破上轨买入，跌破下轨卖出 | 突破行情 |
| MACD | 金叉买入，死叉卖出 | 趋势行情 |
| 海龟交易 | 突破 N 日高点买入 | 趋势跟踪 |
| 均值回归 | 偏离均值过大时反向操作 | 震荡行情 |
| 尾盘突破 | 尾盘买入，次日卖出 | 短线交易 |

---

### 3. 解读回测结果

**策略卡片说明：**

```
┌─────────────────────────────────┐
│ #1 尾盘突破          +0.00% ✅  │  ← 收益率（绿色=正，红色=负）
│ 交易 1 笔              胜率 100% │  ← 交易次数和胜率
│                                 │
│ 最大回撤            最终资金    │
│ 0.00%              ¥100 万      │  ← 风险指标
└─────────────────────────────────┘
```

**关键指标：**

| 指标 | 含义 | 好坏判断 |
|------|------|----------|
| 收益率 | 总收益百分比 | 越高越好 |
| 胜率 | 盈利交易占比 | >50% 较好 |
| 最大回撤 | 最大亏损幅度 | 越低越好 |
| 交易次数 | 总交易笔数 | 适中为宜 |

## 常见问题

### Q1: 前端启动失败，提示端口被占用

**解决：**
```bash
# 查找并关闭占用 3000 端口的进程
lsof -ti :3000 | xargs kill -9
npm run dev
```

### Q2: 后端启动失败，提示 ModuleNotFoundError

**解决：**
```bash
# 安装依赖
cd /Users/mac2025/stock/web/backend
pip install -r requirements.txt

# 或使用 conda Python 启动
/opt/miniconda3/bin/python3 main.py
```

### Q3: 点击「获取数据」后无响应

**可能原因：**
1. 后端未启动 → 检查 8000 端口
2. 股票代码错误 → 确认是 6 位数字
3. 网络问题 → 检查 baostock 服务

**排查步骤：**
```bash
# 检查后端是否运行
lsof -i :8000

# 查看后端日志
# 启动后端时的终端窗口会显示请求日志
```

### Q4: 回测结果显示为 0

**原因：** 数据量不足或策略在该时间段无交易信号

**解决：**
- 延长回测时间范围（如 1 年以上）
- 更换波动较大的股票

### Q5: 手机无法访问

**解决：**
1. 确认手机和电脑在同一 WiFi
2. 检查防火墙设置
3. 确认使用正确的 IP 地址
