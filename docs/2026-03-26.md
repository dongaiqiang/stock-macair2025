### 2. Web 应用开发（React + FastAPI）

#### 2.1 技术选型

考虑到用户后续需要在移动端使用，选择以下技术栈：

| 层级 | 技术 | 理由 |
|------|------|------|
| 后端 | FastAPI + Uvicorn | 高性能异步 API，自动生成 OpenAPI 文档 |
| 前端 | Next.js 14 + React | SSR 支持，良好的移动端体验 |
| 样式 | Tailwind CSS | 快速开发响应式 UI |
| 图表 | Lightweight Charts | TradingView 出品，专业 K 线图 |
| 测试 | Playwright | 微软出品，支持多浏览器 |

1. **路径导入问题**
   ```python

   # 修复后（绝对路径）
   ROOT_DIR = os.path.dirname(os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
   sys.path.insert(0, ROOT_DIR)
   ```

2. **JSON 序列化 NaN/Inf 问题**
   ```python
   # 错误：ValueError: Out of range float values are not JSON compliant: nan

   # 修复：处理 NaN 和 Inf 值
   df_result = df_reset[available_cols].tail(100).copy()
   ```

**当前问题**:

- 前端错误提示为通用消息

**改进方案**:

后端添加详细错误日志：
```python
@app.post("/api/backtest")
def run_single_backtest(request: BacktestRequest):
    try:
        # ... 业务逻辑
    except StockDataError as e:
        logger.error(f"数据获取失败：{e}, stock={request.stock_code}")
        raise HTTPException(status_code=400, detail=f"股票数据获取失败：{str(e)}")
    except StrategyError as e:
        logger.error(f"策略执行失败：{e}, strategy={request.strategy}")
        raise HTTPException(status_code=400, detail=f"策略回测失败：{str(e)}")
    except Exception as e:
        logger.exception(f"未知错误：{e}")
        raise HTTPException(status_code=500, detail="服务器内部错误")
```

前端添加错误边界：
```typescript
try {
  await handleRunBacktest()
} catch (error) {

  alert(`错误：${message}`)
}
```
